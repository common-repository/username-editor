== Description ==

Username Editor is a simple plugin which lets you easily change WordPress usernames which by default is prohibited.

Also, this plugin offers lot more features like ability to let few roles edit their usernames. Also, we have email feature. If set to on. when the user changes his/her username, email will be sent to both the administrator of the site and the user.

= Features =
1. Simple username change.
2. Change username using AJAX.
3. Password Check : Force users to enter their password before changing username.
4. Secure and no-bloat code.
5. Made by developers for developers.
6. Documentation link can be found here [WebsiteGuider](https://websiteguider.com)


== Installation ==

The installation process of this plugin is simple.
1. Upload the zip file or install it from WordPress directory.
2. After installation, activate the theme and you are done

== Frequently Asked Questions ==

= Is there any settings page available? =
Yes, Under Settings > Username Editor, you get many options.

= Does this plugin affects site speed? =
No, this plugin doesn't affect your websites speed.
